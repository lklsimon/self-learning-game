# Learning Games Hub 🎮

一個適合兒童的互動式學習遊戲平台，提供多種有趣的遊戲幫助孩子們學習時間、日期、數字、顏色和月份。

## 遊戲列表

### 🕐 Time Learning Game
學習讀取時鐘！通過有趣的時鐘幫助孩子們學習讀取時間，支援數位和文字兩種格式。

### ⚡ Clock Advanced Game
進階時間計算！學生需要計算未來時間和兩個時鐘之間的時間差，適合小學二年級學生。

### 📅 Month Learning Game
學習英文月份名稱！將中文月份名稱與英文對應起來。

### 📆 Calendar Learning Game
學習使用月曆！查找特定週的日期，並計算未來日期，適合小學二年級學生。

### 🔢 Number Learning Game
學習數字1-100！將中文數字與英文單詞或阿拉伯數字配對。

### 🎨 Color Learning Game
學習英文顏色名稱！將中文顏色名稱與英文翻譯配對。

## 共同特色

- 🎮 **10題一輪遊戲**：完成10題後顯示成績
- 📊 **答題回顧**：點擊1-10按鈕查看每題詳情
- ✅ **即時反饋**：答對/答錯的視覺提示
- 🎯 **進度追進度追蹤**：顯示當前題目和分數
- 🌟 **童趣設計**：可愛的字體和色彩
- 📱 **響應式設計**：支援各種設備尺寸

## 本地運行

直接在瀏覽器中打開 `index.html` 文件即可進入遊戲中心，選擇想要玩的遊戲。
