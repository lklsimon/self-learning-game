# Time Learning Game 🕐

一個適合兒童的互動式時間學習遊戲，通過有趣的時鐘幫助孩子們學習讀取時間。

## 功能特色

- 🎮 **10題一輪遊戲**：完成10題後顯示成績
- 🔄 **雙格式支援**：數位時間和文字時間格式
- 📊 **答題回顧**：點擊1-10按鈕查看每題詳情
- ✅ **即時反饋**：答對/答錯的視覺提示
- 🎯 **進度追蹤**：顯示當前題目和分數
- 🌟 **童趣設計**：可愛的字體和色彩
- 📱 **響應式設計**：支援各種設備尺寸

## 遊戲規則

1. 查看時鐘顯示的時間
2. 從四個選項中選擇正確的時間
3. 回答完10題後查看成績
4. 點擊回顧按鈕查看錯誤題目
5. 點擊「再玩一次」開始新回合

## 本地運行

直接在瀏覽器中打開 `tmp.html` 文件即可開始遊戲。

## 部署

### GitHub Pages

1. 將此文件夾推送到 GitHub 倉庫
2. 在倉庫設置中找到 **Settings** → **Pages**
3. 在 **Source** 部分選擇 `main` 或 `master` 分支
4. 保存後幾分鐘內會自動部署
5. 訪問 `https://your-username.github.io/repo-name/` 即可

### Netlify

1. 訪問 https://app.netlify.com/drop
2. 將整個文件夾拖拽上傳
3. 獲得可分享的 URL
